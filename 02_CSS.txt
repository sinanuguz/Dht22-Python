CSS; Cascading Style Sheets (Basamaklı Biçim Sayfaları)

HTML web sayfalarını oluşturmak için kullanılan standart metin işaretleme dilidir.

CSS ise HTML'e ek olarak metin ve format biçimlendirme alanında fazladan 
olanaklar sunan bir işaretleme dilidir.


label {
  height: 200px;
  width: 50%;
  background-color: powderblue;
}


QPushButton {
     background-color: rgb(225,225,225); 
     border: 2px solid rgb(173,173,173);
     border-radius: 10px;
     color: rgb(50,50,50);
}
